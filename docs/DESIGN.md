### Button

#### Standard button

*original syntax*
```html
<button class="ui button">
  Button
</button>
```

*angular syntax*
```html
<ui-button>Button</ui-button>
```

### Emphasis
*original syntax*
```html
<button class="ui primary button">
  Primary
</button>
```

*angular syntax*
```html
<ui-button class="primary">Primary</ui-button>
```
