## Container

### Using Grids
*semantic syntax*
```html
<div class="ui four column doubling stackable grid container">
  <div class="column">
    <p></p>
    <p></p>
  </div>
  <div class="column">
    <p></p>
    <p></p>
  </div>
  <div class="column">
    <p></p>
    <p></p>
  </div>
  <div class="column">
    <p></p>
    <p></p>
  </div>
</div>
```

*angular syntax*
```html
<ui-container class="four column doubling stackable grid">
    <div class="column">
    <p></p>
    <p></p>
  </div>
  <div class="column">
    <p></p>
    <p></p>
  </div>
  <div class="column">
    <p></p>
    <p></p>
  </div>
  <div class="column">
    <p></p>
    <p></p>
  </div>
</ui-container>
```

### Container Segment
*semantic syntax*
```html
<div class="ui raised very padded text container segment">
  <h2 class="ui header">Dogs Roles with Humans</h2>
  <p></p>
  <p></p>
</div>
```

*angular syntax*
```html
<ui-container class="raised very padded text segment">
  <h2 class="ui header">Dogs Roles with Humans</h2>
  <p></p>
  <p></p>
</ui-container>
```
