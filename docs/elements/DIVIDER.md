## Divider

### Standard
*semantic syntax*
```html
<div class="ui divider"></div>
```

*angular syntax*
```html
<ui-divider></ui-divider>
```

### Vertical
*semantic syntax*
```html
<div class="ui vertical divider">and</div>
```

*angular syntax*
```html
<ui-divider class="vertical">and</ui-divider>
```
